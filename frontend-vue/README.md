# frontend-vue

## Project setup

```
npm install
```

### Compiles and hot-reloads for development, using Vite

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```

### Locally preview production build

```
npm run serve
```

### Lints files

```
npm run lint
```

### Lints and fixes files

```
npm run lint:fix
```

### Formats and saves changes to files

```
npm run format
```
